VentureOS Logo Assets (Flywheel + V)

Files:
- ventureos_logo_icon.svg: primary icon (uses currentColor)
- ventureos_logo_favicon.svg: simplified favicon-safe icon (uses currentColor)
- *_dark.svg / *_light.svg: explicit color variants
- png/: transparent PNG exports at 1024/512/256/128/64/32/16

Tip (web):
Use the currentColor SVG and set color via CSS, e.g.
  <img src="ventureos_logo_icon.svg" style="color:#111" />
or inline SVG:
  svg { color: white; }

Notes:
- The 'icon_bold' variants use thicker strokes for extra legibility at small sizes.
